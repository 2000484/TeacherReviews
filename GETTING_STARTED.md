## Quick Start

### Prerequisites
- Node.js >= 16.0.0
- pnpm >= 7.0.0 (this project uses pnpm)

### Installation & Running

1. **Navigate to the project directory:**
   ```bash
   cd Scramjet-App
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   
   pnpm start
   ```

4. **Open your browser:**
   - Navigate to `http://localhost:8080`
   - The server will be running and ready to use!

## Available Scripts

**Note:** All commands must be run from the `Scramjet-App` directory.

- `pnpm start` - Start the development server
- `pnpm lint` - Check code for linting errors
- `pnpm lint:fix` - Automatically fix linting errors
- `pnpm format` - Format code with Prettier

## Project Structure

- `public/` - Static files and frontend code
- `src/` - Backend server code
- `src/index.js` - Main server entry point
